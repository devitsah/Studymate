StudyMate is a productivity-focused Flutter application designed to help students organize, track, and optimize their study routines effectively. It allows users to create study tasks with subjects, topics, and due dates while providing intelligent reminders before deadlines. The app uses local storage powered by Hive for offline data management and integrates Flutter Local Notifications to ensure you never miss a task.

With a sleek, responsive user interface and real-time task tracking powered by Riverpod state management, StudyMate delivers a seamless experience across devices. The app also includes a statistics dashboard, built with fl_chart, that visually represents completed and pending tasks, allowing users to monitor their study progress easily.

In short, StudyMate isn’t just a task list — it’s your personal study companion that helps you plan smarter, stay consistent, and achieve academic goals efficiently..
